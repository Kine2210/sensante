# Bilan Labs Git - Le Gardien
## 3 choses apprises dans les Labs Git
1. **Les branches** : Travailler sur une branche isolée permet de ne pas perturber le travail des autres membres de l'equipe sur `main`.
2. **Les conflits Git** : Un conflit survient quand deux personnes modifient la même ligne du même fichier. Il faut lire les marqueurs (`<<<<<<<`, `=======`, `>>>>>>>`), choisir la bonne version, supprimer les marqueurs, puis faire `git add` + `git commit`.
3. **Le workflow complet** : La chaîne branche → commit → push → Pull Request → review → merge → tag est la base d'une collaboration propre et organisée en équipe.
